The possible values for position are: static (default), relative, absolute, and fixed.

  • static - default
  • fixed - does not move regardless of scroll
  • relative
  • absolute

Ex 2: setting the second box to have an offset from the top-right

        .offset-tr {
          position: absolute;
          right: 20px;
          top: 20px;
        }
As we see, it ends up at the top of the page. It lacks a parent to derive it's relative positioning from. Therefore it's positioned based on the body.

Ex 3: box 2 with offset and parent set to relative

        .offset-tr {
          position: absolute;
          right: 20px;
          top: 20px;
        }
        .relative {
          position: relative;
        }
In this case we've also wrapped the whole group in a div with .relative set. The position of the 'absolute'ly positioned element is derived from the parent 'relative'ly positioned element.
