box-shadow

box-shadow is similar to text-shadow except it sets shadow for the entire box. It also accepts a couple of optional extra parameters. The first is spread which controls the size of the shadow. Negative values are allowed. Inset can be placed (optionally) after color and changes the shadow from an outer shadow to an inner shadow.

        box-shadow: 2px 2px 2px 8px rgba(255,0,0, .5), inset;
        box-shadow: x y blur* spread* color**, inset*;

Note that in the above '*' denotes an optional parameter.

box-shadow: 3px 5px 0px 1px rgba(0,0,255,1);
