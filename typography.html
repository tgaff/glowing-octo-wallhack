text properties

Next we'll take a look at text properties. These allow us to align, decorate, indent, transform and control the spacing of text.

text-align, text-decoration and text-indent

text-align can be set to left, right, center, justify and inherit

text-decoration can be set to none, underline, overline, line-through and inherit.

text-indent only indents the first line of text in an element. All common length values are supported. Here we use a 40px indent.

  text-align: right;
        text-decoration: underline;
        text-indent: 40px;

text-shadow

text-shadow takes parameters for:

  • horizontal-offset
  • vertical-offset
  • blur-radius
  • color and opacity
By setting negative values for the offsets the shadow can be moved upwards and to the left. 

text-shadow: 4px 8px 2px rgba(255,0,0, .5);

box-shadow

box-shadow is similar to text-shadow except it sets shadow for the entire box. It also accepts a couple of optional extra parameters. The first is spread which controls the size of the shadow. Negative values are allowed. Inset can be placed (optionally) after color and changes the shadow from an outer shadow to an inner shadow.

        box-shadow: 2px 2px 2px 8px rgba(255,0,0, .5), inset;
        box-shadow: x y blur* spread* color**, inset*;

Note that in the above '*' denotes an optional parameter.

box-shadow: 3px 5px 0px 1px rgba(0,0,255,1);
